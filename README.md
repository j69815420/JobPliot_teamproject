## 1. 프로젝트 소개
취업을 준비하는 사용자가 실전 면접처럼 연습하고 실시간으로 AI로부터 구체적인 피드백을 제공받아 안드로이드 앱을 구현한 팀 프로젝트.
### 개발 기간
- 25.03~25.11 (8개월)
## 2. 주요 기능
- 사용자의 음성 및 텍스트 답변을 분석하여 실시간으로 AI 기반 점수 및 피드백 제공
- AI 또는 사용자로부터 선택된 분야로부터 질문 제공
- AI로부터 받은 점수 및 피드백 조회
- 자기소개서 작성 및 파일 첨부 후 AI 기반 피드백 제공
